# Smart AI Railway Reservation System

A modern, responsive web application that simulates an intelligent railway ticket booking platform with AI-powered features.

## Features

### AI-Based Seat Allocation
- Automatically assigns seats based on user preferences (window, lower berth, etc.)
- Ensures group bookings are seated together
- Prioritizes elderly and special needs passengers

### Smart Waiting List Prediction
- Shows probability of ticket confirmation (High / Medium / Low)
- Uses mock AI logic based on demand, season, and historical booking trends

### Auto Seat Upgrade
- Automatically upgrades users to higher classes if seats become available
- Displays upgrade notifications to the user

### Voice-Based Booking
- Allows users to input booking details using voice commands
- Converts speech to text and auto-fills booking fields
- Uses Web Speech API for voice recognition

### Face Recognition Verification (Simulation)
- Simulates face verification during login or booking confirmation
- Shows success/failure messages

## Core Functionalities
- User registration and login system
- Train search and availability display
- Ticket booking interface
- Booking confirmation page with ticket details

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Styling:** CSS with modern design
- **AI Simulation:** JavaScript logic
- **Voice Recognition:** Web Speech API

## How to Run
1. Open `index.html` in a modern web browser
2. Register or login with any username/password
3. Browse available trains and book tickets
4. Try voice booking (requires microphone permission)
5. Test face recognition simulation

## Browser Requirements
- Modern browser with Web Speech API support (Chrome, Edge, Safari)
- Microphone access for voice features
